wsexampledata
-------------------------

Ejemplo de datos para servir mediante REST, con el objetivo de realizar pruebas de servicios específicos.

Archivo de datos tomado de https://www.briandunning.com/sample-data/. No se incluye por respeto a los que lo proveen,
pero se puede descargar de dicha URL y realizar los ajustes de formato que se describen a continuación.

Se cambió para que tuviera los siguientes nombres de campos (adicionando el *username* con el valor del *email*):
firstname;lastname;institution;address;city;county;state;zip;phone1;phone2;email;web;username

Además, se cambia "," (coma) por ";" (punto y como) como separador de campos y se quitan las comillas entre los valores.
